## 一、课程定位：先讲清对象、边界和工程后果

从失效机理、制造过程、供应链变化和寿命应力建立可靠性闭环。

## 二、概念骨架与必须讲透的知识

- **失效机理**：EOS、ESD、焊点疲劳、腐蚀、过热和机械冲击会留下不同物理痕迹。
- **过程窗口**：SMT、三防漆、老练、振动和来料检验都有可量化工艺窗口。
- **应力筛选**：HALT/HASS/Burn-in 通过温度、振动、电压和负载加速暴露薄弱点。
- **供应链变更**：PCN/EOL、替代料和降本会改变电气、机械、热和认证边界。
- **数据闭环**：量产质量依赖不良分类、批次追溯、Pareto 和纠正预防措施。

## 三、公式模型、参数计算与适用边界

- **失效机理**：FA 先保护现场和样品，再做无损到有损分析。
- **过程窗口**：温度曲线、焊膏、湿敏等级、涂覆厚度和固化条件要记录。
- **应力筛选**：应力要高于使用条件但不引入非真实失效机理。
- **供应链变更**：替代料需比较参数、封装、工艺、寿命和测试证据。
- **数据闭环**：用良率、DPPM、CPK 和重复故障率跟踪效果。

## 四、嵌入式硬件中的典型场景

- 多层 PCB 微裂纹常在热循环或振动后出现间歇性开路，常温静态测试不一定能复现。
- 三防漆在高阻节点附近过厚或污染会引入漏电，清洁度比涂覆本身更关键。
- BOM 降本不能只比较价格，要验证温漂、寿命、封装兼容和供应稳定性。
- RMA 退回件要把客户现场电源、接地、环境和操作记录纳入 FA。

## 五、观察、测量与排错方法

- FA 流程按外观、X-ray、显微、IV 曲线、开封和截面逐步推进。
- 制造问题通过首件确认、炉温曲线、AOI/X-ray 和抽检数据定位。
- 环境应力测试记录样品数量、应力曲线、失效时间和失效模式。
- 供应链替代要做 A/B 样品对比、边界测试和小批量试产。

## 六、课堂练习与交付

课堂交付：**国际第三方机构合规认证资料包（Technical File）可靠性与质量闭环记录**。

1. 建立一个失效样品的 FA 流程和证据清单。
2. 为一种替代料列出电气、热、机械和认证风险。
3. 设计一条 Burn-in 或 HALT 应力曲线。
4. 用 Pareto 方法分析一批量产缺陷并给出整改优先级。

## 七、本课检查表

- [ ] 关键参数都有数值、单位、温度和工况边界。
- [ ] 电流路径、信号路径或能量路径能从图纸追到实物。
- [ ] 启动、稳态、瞬态、异常和恢复状态都已覆盖。
- [ ] 测试方法写清仪器带宽、夹具连接和合格判据。
- [ ] 设计取舍能解释成本、可靠性、可制造性和维护影响。
