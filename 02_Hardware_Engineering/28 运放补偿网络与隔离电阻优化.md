### 关键公式与参数

- 电阻基础关系为 $V=IR$、$P=I^2R$；精密应用还要校核 TCR、耐压、噪声和封装热阻。
- 运放闭环精度由开环增益、反馈网络、输入失调、偏置电流、GBW、SR 和输出摆幅共同决定。
- 高速 PCB 判断重点是参考平面连续、阻抗一致、回流路径短、过孔 Stub 可控和 PDN 无尖锐反谐振。
- 闭环带宽近似 $f_{BW}\approx \frac{GBW}{A_v}$。
- 正弦满功率带宽约 $f_{max}=\frac{SR}{2\pi V_{pk}}$。
- 热噪声按 $e_n=\sqrt{4kTRB}$ 初估，高阻反馈网络尤其敏感。

### 产品场景

- 用于传感器前端、弱信号放大、偏置生成、滤波、差分采样和模拟保护。
- 适合建立从误差源、稳定性到 Layout 的完整模拟信号链思维。
- 本课交付物对应的典型场景：交付：10nF 纯容性负载下运放稳定性整改设计包

### 设计与 Layout 要点

- 先确定输入范围、共模范围、输出摆幅和供电，再选择拓扑和运放。
- 高阻节点要短、干净、远离开关节点；反馈网络靠近运放输入脚。
- 容性负载、长线和 ADC 采样电容前通常需要隔离电阻或缓冲。
- 将「隔离电阻 $R_{iso}$ 计算、输出反馈补偿、噪声增益（NG）塑形、开环阻抗仿真。」转成原理图注释、器件参数、网名、测试点和评审 Checklist。

### 测试验证

- 用低噪声信号源、示波器和频响测试确认增益、带宽、相位和过冲。
- 直流精度测试记录零点、满量程、温漂和电源变化。
- 怀疑自激时观察输出噪声底、频谱尖峰和阶跃响应振铃。
- 测试记录至少包含样机版本、仪器型号、连接方式、环境条件、波形截图和判定结论。

### 典型故障与排查

- 把虚短虚断当成无条件成立，忽略开环增益、输入输出摆幅和反馈方向。
- GBW、SR 或相位裕度不足，静态正确但动态失真或振荡。
- 电阻比例、偏置电流路径或 REF 引脚处理不当，导致精度无法闭环。
- 排查时按「现象 → 假设 → 测量 → 结论 → 预防」的顺序闭环，不跳过证据链。
