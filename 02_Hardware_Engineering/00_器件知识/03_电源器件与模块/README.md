# 电源器件与模块

覆盖电源转换、分配、保护、充电和电源树管理相关的芯片与成品模块。

## 现有文档

- [线性稳压器](<线性稳压器.md>)

## 覆盖范围

- LDO、线性稳压器、基准型稳压器和低噪声电源。
- Buck、Boost、Buck-Boost、反激、LLC、PFC 控制器及集成功率芯片。
- AC-DC、DC-DC、隔离电源、POL 和 DIN 导轨/封装电源模块。
- PMIC、负载开关、理想二极管、热插拔、eFuse 和电源时序器。
- 电池充电、Fuel Gauge、均衡、保护和 BMS AFE。

## 建议文章

- AC-DC 电源选型。
- DC-DC 电源模块选型。
- LDO 与开关稳压器选型边界。
- 负载开关、热插拔与 eFuse 比较。

## 选型重点

- 输入/输出范围、功率、效率、纹波、瞬态、启动、控制模式和环路要求。
- 隔离等级、耐压、漏电流、EMC、安规认证、海拔和降额曲线。
- 保护功能、热设计、寿命、封装、供应链和外围器件约束。
