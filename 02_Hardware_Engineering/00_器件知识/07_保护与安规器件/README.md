# 保护与安规器件

覆盖过流、过压、浪涌、静电、温度和隔离安全相关的器件及其协调配合。

## 覆盖范围

- 熔断器、PTC/PPTC、NTC 和温度保险丝。
- TVS、ESD 阵列、MOV、GDT、压敏器件和浪涌抑制模块。
- 隔离槽相关安规器件、Y/X 电容和安全放电电阻。
- 过压/欠压监控、浪涌控制器和保护控制芯片。

负载开关、热插拔和 eFuse 统一归入 [电源器件与模块](<../03_电源器件与模块/README.md>)。

## 选型重点

- 正常工作边界、触发/熔断特性、钳位电压、残压、能量和重复脉冲能力。
- 保护器件之间的动作时序、能量分配、退耦阻抗和失效模式。
- 漏电流、结电容、老化、温升、认证、污染等级和短路后安全状态。
