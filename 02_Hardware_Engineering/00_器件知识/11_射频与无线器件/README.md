# 射频与无线器件

覆盖射频信号的产生、放大、切换、滤波、匹配、收发和辐射。

## 覆盖范围

- LNA、PA、射频开关、衰减器、混频器、VCO 和频率合成器。
- SAW/BAW/LC 滤波器、双工器、功分器、耦合器和 Balun。
- 天线、匹配网络、射频连接器和同轴组件。
- Wi-Fi、Bluetooth、GNSS、蜂窝、LoRa、Zigbee 和其他无线模组。

## 选型重点

- 频段、增益、噪声系数、线性度、P1dB、IP3、隔离度和插入损耗。
- 阻抗、驻波、功率容量、谐波、相位噪声、匹配和 PCB 材料损耗。
- 认证、天线净空、共存干扰、射频布局、屏蔽和量产校准。
