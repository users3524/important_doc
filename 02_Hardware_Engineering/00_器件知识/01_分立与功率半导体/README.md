# 分立与功率半导体

覆盖整流、开关、放大和功率变换所使用的分立半导体及功率器件。

## 现有文档

- [二极管](<二极管.md>)
- [三极管](<三极管.md>)
- [MOSFET](<MOSFET.md>)

## 覆盖范围

- PN、肖特基、快恢复和稳压二极管。
- BJT、MOSFET、IGBT、晶闸管和 Triac。
- SiC MOSFET、SiC 二极管和 GaN 功率器件。

TVS 和其他浪涌防护器件统一归入 [保护与安规器件](<../07_保护与安规器件/README.md>)。

## 选型重点

- 电压、电流、导通损耗、开关损耗、结电容和反向恢复。
- 栅极/基极驱动、SOA、雪崩能力、短路耐量和寄生参数。
- 结温、热阻、封装爬电、并联均流和失效模式。
