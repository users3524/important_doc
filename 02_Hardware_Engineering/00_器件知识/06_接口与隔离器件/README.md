# 接口与隔离器件

覆盖板内、板间和现场总线的物理层接口，以及跨电气域的信号隔离。

## 覆盖范围

- RS-232、RS-422/485、CAN/CAN FD、LIN 和工业现场总线收发器。
- USB、Ethernet PHY、HDMI、MIPI 和 SerDes 接口器件。
- I2C/SPI/UART 扩展、桥接、隔离和电平转换器件。
- 数字隔离器、光耦、隔离栅极驱动和隔离式收发器。

## 选型重点

- 速率、传播延迟、抖动、共模范围、节点数、终端和失效保护。
- 隔离耐压、工作电压、CMTI、爬电距离、寿命和隔离电源需求。
- ESD/浪涌、结电容、EMC、协议兼容性和连接器/线缆条件。
