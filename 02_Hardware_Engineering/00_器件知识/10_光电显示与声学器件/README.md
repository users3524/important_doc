# 光电显示与声学器件

覆盖光的产生/检测、视觉显示和声音输入输出器件。

## 覆盖范围

- 指示 LED、照明 LED、红外发射管、激光器和 LED 驱动器。
- 光电二极管、光电三极管、光敏电阻和环境光接收器。
- 数码管、LCD、OLED、电子纸、背光和触摸显示模组。
- 蜂鸣器、扬声器、麦克风和音频换能器。

光耦作为隔离器件统一归入 [接口与隔离器件](<../06_接口与隔离器件/README.md>)。

## 选型重点

- 光强、波长、视角、色温、响应度、暗电流、寿命和热衰减。
- 显示接口、刷新、功耗、亮度、温区、可视性和烧屏风险。
- 声压级、频响、谐振频率、灵敏度、驱动功率和结构声腔。
