## 归类边界

- 现成 AC-DC/DC-DC 电源模块及其选型方法放入 `03_电源器件与模块`；可直接用 EDA 工具打开的参考工程放入上一级 `01_模块整理`。
- TVS、MOV、保险丝等按保护协调维护在 `07_保护与安规器件`，不在二极管或被动器件目录复制正文。
- ADC/DAC 和模拟前端按信号链维护在 `02_模拟与信号链器件`；传感器本体放入 `08_传感器与采集器件`。
- 项目专用的具体型号比较和最终 BOM 结论归档到 `04_Project`；本目录保留可复用的方法、参数解释和跨型号比较维度。
- 测试步骤和故障复盘分别放入上一级 `03_调试技巧`，器件文档只保留与该器件直接相关的验证入口。

## 器件文档基线

一篇可复用的器件文章至少应说明：

- 功能、工作原理、典型拓扑和不适用场景。
- 关键参数的定义、测试条件、温度/频率依赖和相互制约。
- Absolute Maximum、Recommended Operating Conditions、降额和 SOA。
- 封装、热设计、PCB 布局、寄生参数和装配限制。
- 常见失效模式、保护策略和调试测量方法。
- 替代料比较维度、供应状态、数据手册版本和事实来源。
